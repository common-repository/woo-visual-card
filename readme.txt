Show a visual card in WooCommerce checkout page.

== Description ==
= Show a configurable visual card in WooCommerce checkout page for the following payment gateways: =
* BusinessPay

If you need run it with another payment gateways, please let us know.

= Compatibilidade =
O plugin verifica a versão do PHP, do Wordpress e do WooCommerce para determinar se é compatível com a versão atual do plugin.
Sugestões serão bem-vindas.

== Installation ==
= How to install: =
* Upload the files to the wp-contents/plugins directory of your Wordpress installation;
* Activate the plugin;
* The visual card will appear in checkout page from BusinessPay Gateway;
